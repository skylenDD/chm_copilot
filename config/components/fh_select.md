#### 下拉框 (fh_select)
- **版本**: 1.0.0
- **分类**: form
- **图标**: icon-select
- **属性**:
  - `label`: 标签 (string, 默认值: "选择项")
  - `options`: 选项列表 (array, 每项包含 label 和 value)
  - `required`: 是否必选 (boolean, 默认值: false)
  - `defaultValue`: 默认值 (string, 默认值: "")
- **事件**:
  - `onChange(ctx)`: 选择变化事件，ctx 包含组件上下文信息，默认实现：`console.log('onChange', ctx)`
